# Project Brief

## Vision
Un site hybride pour Thomas Aubert : d'un côté, une plateforme vitrine de "personal branding" pour se positionner professionnellement (ton rationnel, direct, élégant). De l'autre, un espace d'administration minimaliste dédié exclusivement à la gestion de son calendrier de compétitions jusqu'à fin août 2028.

## Design
Le design doit refléter la précision, la droiture, la concentration et le haut niveau.
- Style : Épuré, professionnel, contrasté (Dark Navy et Beige).
- --background: 40 33% 96% (Sable clair / Beige)
- --foreground: 222 47% 11% (Bleu Nuit profond)
- --primary: 222 47% 11% (Bleu Nuit)
- --primary-foreground: 40 33% 96% (Sable)
- --accent: 43 74% 49% (Or / Bronze - rappel médailles)
- --muted: 40 20% 90%
- Fonts: 'Outfit' pour les titres (affirmé, géométrique), 'Inter' pour le corps de texte (lisibilité, rationalité).

## Features
- Espace public : Vitrine, Parcours pro/sportif, Palmarès, Matériel, Vision.
- Espace privé (Backend) : Accès sécurisé à un calendrier de compétitions restreint jusqu'au 31 Août 2028.
- Gestion des événements : Ajout, suppression et édition du programme détaillé pour chaque événement.
- Export : Génération et téléchargement de fichiers .ics pour intégration dans les agendas personnels.
