## Notes
Permettre d'enrichir une compétition avec son programme détaillé lorsqu'il est connu, et de l'exporter pour les agendas personnels.

## Checklist
- [ ] Interface (modal ou page dédiée) pour éditer une compétition existante et y ajouter du texte détaillé (le programme)
- [ ] Générateur de fichier `.ics` formaté correctement incluant le titre, les dates, le lieu et la description
- [ ] Bouton "Exporter vers mon agenda" sur chaque événement pour déclencher le téléchargement

## Acceptance
- L'utilisateur peut ajouter et consulter le programme spécifique d'un événement
- Le téléchargement du fichier ICS génère un fichier valide et lisible par les applications d'agenda standard (Apple, Google, Outlook)
